# 基金看板 Pro (Fund Dashboard Pro)

这是一个为移动端优化的轻量级基金看板工具，旨在为个人投资者提供直观、实时的基金估值监控、持仓管理及止盈止损提醒功能。项目采用纯前端技术栈，支持直接通过 GitHub Pages 或 Vercel 进行快速部署。

## 🚀 核心特性

- **实时估值监控**：利用 JSONP 技术直连天天基金数据源，实时更新持仓估值与收益。
- **移动端增强体验**：
  - **沉浸式 UI**：采用现代化的卡片式设计和深色模式适配。
  - **交互优化**：支持列表长按拖拽排序（基于 Sortable.js）。
  - **图表交互**：集成 Chart.js，支持手势缩放和平移查看历史净值趋势。
- **持仓智能提醒**：
  - **止盈/抄底提醒**：支持自定义目标价，当实时估值触及目标时自动显示提醒标签（🎯止盈 / 💰机会）。
  - **成本线对标**：在历史趋势图中自动绘制持仓成本线与目标提醒线。
- **数据隐私**：所有配置信息（基金代码、持仓份额、成本等）均存储在用户的浏览器 `localStorage` 中，不经过任何后端服务器，保护投资隐私。

## 🛠 技术栈

- **前端框架**：原生 HTML5 / CSS3 (Flexbox & CSS Variables) / JavaScript (ES6+)。
- **数据可视化**：[Chart.js](https://www.chartjs.org/) v4.4.1。
- **手势库**：[Hammer.js](https://hammerjs.github.io/) & [chartjs-plugin-zoom](https://www.chartjs.org/chartjs-plugin-zoom/)。
- **拖拽排序**：[Sortable.js](https://sortablejs.com/)。
- **数据源**：天天基金公开 API。

## 📦 快速开始

### 部署方式
由于项目是静态网页，您可以直接通过以下方式访问：
1. **GitHub Pages**：将代码推送至仓库后，在 Settings -> Pages 中开启服务。
2. **本地运行**：直接使用浏览器打开 `index.html` 即可使用。

### 使用指南
1. **添加基金**：点击右下角浮动按钮（+），输入 6 位基金代码。
2. **配置持仓**：输入您的持有份额和持仓成本，系统将自动计算持仓市值和今日盈亏。
3. **查看详情**：点击基金卡片可进入趋势面板，查看 1 月、3 月、1 年及全量历史净值。
4. **管理列表**：
   - **排序**：在列表页长按卡片并上下拖动可调整显示顺序。
   - **编辑/删除**：点击卡片右下角的齿轮图标（⚙）进入配置页面。

## 📂 文件结构

- `index.html`: 主入口文件，包含基金列表、实时渲染逻辑及拖拽排序功能。
- `fund.html`: 移动增强版原型，包含核心数据逻辑与图表组件。
